How projects move through the core

2

Standard QC checks, run notes, and data delivery in agreed formats.

What you receive

  • Run summary (inclusions/exclusions, deviations, notes)
  • Raw data exports (system-dependent)
  • Processed metrics table (CSV) + basic plots
  • Optional consultation on interpretation and next steps

What we need from you

  • Clear hypothesis and primary endpoint
  • Strain/age/sex and handling history
  • Any drug dosing schedule and washout constraints
  • Requested timeline and cohort availability
